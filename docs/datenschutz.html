Welche Daten wir speichern

  • Lernfortschritt (Lektionen, Buchstaben, Wörter)
  • Lesezeichen und Leseposition im Quran
  • Einstellungen (Schriftgröße, Theme, Rezitator)
  • Streak- & Tagesziel-Daten
  • Quiz-Ergebnisse und Übungsfortschritt

Alle Daten werden ausschließlich lokal auf deinem Gerät gespeichert (SharedPreferences & Hive). Es findet keine Übertragung an externe Server statt.

Audio-Daten

Für die Quran-Rezitation werden Audio-Dateien direkt von everyayah.com gestreamt. Dabei wird lediglich die IP-Adresse deines Geräts an den Server übertragen. Wir speichern keine personenbezogenen Daten auf diesem Server.

Google Fonts

Die App lädt Schriftarten über Google Fonts. Dabei wird eine Verbindung zu Google-Servern hergestellt und deine IP-Adresse übertragen. Weitere Informationen: fonts.google.com/about

Benachrichtigungen

Wenn du die Erinnerungsfunktion aktivierst, werden lokale Benachrichtigungen geplant. Diese werden ausschließlich auf deinem Gerät verarbeitet – es werden keine Daten an Push-Dienste gesendet.
